# Template GitHub Pages

Este é um template modelo para hospedar no GitHub Pages.

## Como usar

1. Crie um repositório no GitHub
2. Execute os comandos abaixo nesta pasta:

```bash
git init
git add .
git commit -m "Primeiro commit"
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/NOME-DO-REPO.git
git push -u origin main
```

3. Vá em Settings > Pages no seu repositório
4. Selecione a branch `main` e clique em Save
5. Acesse: `https://SEU-USUARIO.github.io/NOME-DO-REPO/`

## Estrutura

- `index.html` - Página principal
- `style.css` - Estilos
- `script.js` - JavaScript
